### Expense Tracker

This is a simple expense tracker API built using PHP and Laravel. It allows users to add, view, and delete expenses.

**Features**

- Add new expenses with a description and amount
- View a list of all expenses with their descriptions and amounts
- Delete expenses from the list
- Edit expenses to update their descriptions or amounts
- Filter expenses by date range
- Sort expenses by date, amount, or description
- Create Goals and track progress towards them
- Create Budgets and track spending against them
- View a summary of expenses and budgets
- Generate reports on expenses and budgets
